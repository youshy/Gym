# v4 Backup — Hotel-Room Bodyweight (delegation Tue–Thu)

**When:** delegation days, Tue through Thu inclusive. No gym, small hotel room, **one session/day.**
**Goal:** hold what v4 built — push, legs, pull, core — with zero equipment. Not to progress, just not lose ground for 3 days.
**Kit (optional):** a loaded backpack (books/water bottles) turns bodyweight rows and squats into real load. A towel + a solid door = pull work. That's it.

No load to chase, so intensity comes from **reps to near-failure, slow tempo (3s down), and harder variations** instead of heavier weight.

---

## The 3 Days

| Day | v4 equivalent | Focus | Run (if you can get outside) |
|-----|---------------|-------|------------------------------|
| **Tue** | Bench + Press | **Push** — chest, shoulders, triceps | 4–5km easy |
| **Wed** | rest + long run | **Legs** — quads, glutes, hinge | 8–12km easy (or just this session) |
| **Thu** | Deadlift + pull | **Pull + posterior** — back, hamstrings, grip | 3km easy, flat |

Order every day: warm-up (~5 min) → main work → 1 core finisher → run if available.
Running is optional here — if you can step outside, keep it easy Z2 per v4. If not, the session stands alone.

---

## Tue — Push

Warm-up: arm circles, band pull-aparts (or towel), 1×15 easy pushups.

| Move | Sets × reps | Notes |
|------|-------------|-------|
| Pushups (feet elevated on bed for harder) | 4 × AMRAP-2 | 3s lowering. Elevate feet once flat pushups feel easy. |
| Pike pushups (hips high, head to floor) | 3 × 8–12 | This is your overhead-press stand-in. |
| Chair/bed dips (hands behind you on edge) | 3 × 10–15 | Triceps + lower chest. |
| Diamond pushups | 2 × AMRAP | Triceps finisher. |

**Core:** plank 3 × 30–60s (anti-extension, v4 Mon slot).

---

## Wed — Legs

Warm-up: leg swings, bodyweight squats ×15, ankle rocks.

| Move | Sets × reps | Notes |
|------|-------------|-------|
| Bulgarian split squat (rear foot on bed) | 3 × 10–12 / leg | The main event — brutal, needs no load. Hold backpack for more. |
| Squats, 3s down + 1s pause | 4 × 20 | Or pistol progression if 20 is easy. |
| Single-leg RDL (hinge, backpack optional) | 3 × 10 / leg | Hamstring/glute, balance. |
| Calf raises off a step/edge | 3 × 20 | |
| Glute bridge (single-leg for more) | 3 × 15 | |

**Core:** hanging leg raises aren't available → **lying leg raises 3 × 15** (flexion, v4 Fri slot).

---

## Thu — Pull + Posterior

The hard one to replicate without a bar. Use the door + towel and/or backpack.

Warm-up: cat-cow, glute bridges, band/towel pull-aparts.

| Move | Sets × reps | Notes |
|------|-------------|-------|
| Towel door rows (towel around a solid door handle, lean back, pull) | 4 × 12–15 | Your row/pullup stand-in. Lean more horizontal = harder. |
| Backpack bent-over rows | 3 × 12–15 | Load it heavy. |
| Superman holds | 3 × 20–30s | Posterior chain, spinal erectors. |
| Reverse snow-angels / prone Y-T-W (face down) | 3 × 12 | Rear delts + face-pull replacement — v4's daily shoulder insurance. |
| Backpack single-leg RDL | 3 × 10 / leg | Hamstring + grip. |

**Core:** suitcase carry with backpack, 3 × 20 steps/side (anti-lateral flexion + grip, v4 Thu slot).

---

## Rules

- **One session/day only** — if a day gets eaten by work, drop it, don't double up.
- Priority if you're short on time: **Bulgarian split squats, towel rows, pushups.** Those three cover the most.
- Keep runs genuinely easy or skip them — recovery on the road is worse than at home.
- Back on v4's real schedule Friday. Treat this as maintenance, expect the first heavy day back to feel a touch off. Normal.
