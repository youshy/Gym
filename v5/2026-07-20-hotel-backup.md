# v5 Backup — Zero-Equipment Power (travel / hotel room)

**When:** travel days, no gym, one session/day. **Assumes literally no equipment** — no backpack, no towel, no bands. Hotel furniture (bed, chair, desk, wall, floor) is *fair game but never required*.
**Goal:** hold v5's pillars — **explosive power**, push, legs, pull, core — for a few days. Not to progress, just not lose ground.

**Why this works:** v5's priority is *power*, and plyometrics need nothing. Jumps and plyo pushups translate to the road perfectly, so the athletic centerpiece is fully preserved. Intensity comes from **max-intent explosiveness, reps to near-failure, slow tempo (3s down), and harder variations** — not load.

**The one honest gap:** pulling. With zero kit there's no real pullup/row stand-in beyond furniture and isometrics. Pull days here are **maintenance** — expect the first bar session back to feel off. Normal.

---

## The 3 Days

| Day | v5 equivalent | Focus | Run (if you can get outside) |
|-----|---------------|-------|------------------------------|
| **A** | Tue/Sat (bench + upper plyo) | **Push + upper power** | 4–5km easy |
| **B** | **Fri (cleans + jumps)** | **Legs + jumps + Oly patterning** — the centerpiece | 3km easy, flat (legs are cooked) |
| **C** | Thu (deadlift + pull) | **Pull + posterior** (maintenance) | 4–5km easy |

Order every day: **warm-up → explosive work first (fresh) → strength → core → run if available.**
Cycle A→B→C→A… across your trip. If a day gets eaten by work, drop it — don't double up.
Running optional; keep it genuinely easy or skip (recovery on the road is worse than home).

---

## Day A — Push + upper power

**Warm-up:** arm circles, scap pushups ×10, thoracic rotations/side, 1×15 easy pushups.

| Move | Sets × reps | Notes |
|------|-------------|-------|
| **Clap / explosive pushups** (plyo — do first) | 4–5 × 3–5 | Max intent, full rest. This is v5's upper plyo. Push hard off the floor. |
| Pushups, feet elevated on bed | 4 × AMRAP-2 | 3s lowering. Elevate feet once flat feels easy. |
| Pike pushups (hips high, head to floor) | 3 × 8–12 | Overhead-press stand-in. |
| Chair/bed dips (hands behind on edge) | 3 × 10–15 | Triceps + lower chest. |
| Diamond pushups | 2 × AMRAP | Triceps finisher. |

**Core:** hollow-body hold **or** plank 3 × 30–60s (anti-extension — your ab-wheel stand-in).

---

## Day B — Legs + jumps + Oly patterning (the centerpiece)

**Warm-up:** leg swings, bodyweight squats ×15, ankle rocks, deep-squat holds → **jump ramp: pogos → tuck jumps** (a few easy, building height).

**Oly skill patterning first (no load — groove the positions):**
| Move | Sets × reps | Notes |
|------|-------------|-------|
| Overhead-squat pattern (arms locked high, deep squat) | 2 × 5 | Snatch-catch mobility + position. |
| Jump-shrug / high-pull pattern (hinge → violent extension → shrug) | 3 × 5 | The clean/snatch pull, unloaded. Explode tall, stay crisp. |

**Jump work (plyo — fresh, max intent, full recovery, land soft):**
| Move | Sets × reps | Notes |
|------|-------------|-------|
| Jump squats | 4 × 5 | Reset each rep. Quality over fatigue. |
| Broad jumps | 3 × 3 | Space permitting — measure, try to beat it. |
| Tuck jumps | 3 × 5 | Knees to chest, soft landings. |
| Pogo hops (stiff ankles, minimal knee) | 3 × 20 | Tendon stiffness / running support. |

**Strength:**
| Move | Sets × reps | Notes |
|------|-------------|-------|
| Bulgarian split squat (rear foot on bed) | 3 × 10–12 / leg | Brutal, needs no load. |
| Squats, 3s down + 1s pause | 4 × 20 | Or pistol progression if 20 is easy. |
| Single-leg RDL (bodyweight, slow) | 3 × 10 / leg | Hamstring/glute + balance. |
| Calf raises off a step/edge | 3 × 20 | |

**Core:** lying leg raises 3 × 15 (flexion — hanging-leg-raise stand-in).

---

## Day C — Pull + posterior (maintenance)

The hard one with zero kit. Do what the room allows; accept it's holding, not building.

**Warm-up:** cat-cow, glute bridges, big arm swings / pull-apart mimics, prone scap squeezes.

| Move | Sets × reps | Notes |
|------|-------------|-------|
| **Inverted rows under a solid desk/table** | 4 × AMRAP | *If the room has one.* Pullup/row stand-in — the best pull available. Feet out = harder. |
| **Doorway isometric rows** (grip both sides of a solid door frame, lean back, pull hard) | 5 × 20s | Fallback if no desk. All-out isometric — keeps the pattern firing. |
| Prone Y-T-W / reverse snow-angels (face down) | 3 × 12 | Rear delts + face-pull replacement — v5's daily shoulder insurance. |
| Superman holds | 3 × 20–30s | Posterior chain, spinal erectors. |
| Single-leg RDL (slow) | 3 × 10 / leg | Hamstring + balance. |
| *Optional:* broad jumps / bounds | 3 × 3 | If space — keep a little power in on pull day too. |

**Core:** side plank 3 × 30–45s / side (anti-lateral flexion).

---

## Rules

- **One session/day only.** Short on time? Priority moves that cover the most: **jump squats + Bulgarian split squats (Day B), clap + elevated pushups (Day A), inverted/iso rows (Day C).**
- **Explosive work always goes first**, while fresh — same as v5. Never grind a jump; if height drops, that set was the last.
- Keep runs genuinely easy or skip them.
- Back on v5's real schedule the day you're home. Treat this as maintenance; the first heavy/Oly session back will feel a touch off. Normal.
