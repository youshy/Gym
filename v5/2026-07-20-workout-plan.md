# Workout Plan v5 — Athletic Power (Oly + Plyo) over a 5/3/1 Base + Easy Running

**Goal:** Shift the center of gravity from "how heavy can the bar go" to **"how much force can I produce fast."** Strength (5/3/1) becomes a *sustainable base* that stops escalating aggressively; snatch, cleans, and plyometrics become the priority. Running stays as easy Z2 conditioning, unchanged from v4.

**Why the shift:** the v4 5/3/1 grind becomes a nightmare once weights scale and it's stacked on daily running. The Olympic lifts fix this structurally — *you can't grind a snatch.* Load climbs, quality stays explosive.

**Schedule:** 5 lift days (Mon, Tue, Thu, Fri, Sat) + running 7 days/week. Rest from lifting: Wed, Sun (big run days).

**Calculator:** live weights live in [`v5-calculator.html`](v5-calculator.html) — update TMs there as you go; don't hand-maintain numbers in this doc.

---

## The Two Structural Moves (vs v4)

1. **5's PRO replaces heavy AMRAP.** Same 5/3/1 engine and TM progression, but every top set is a clean **5 reps at the prescribed %, no AMRAP.** This removes the max-effort grind that ate your nervous system, leaving CNS for the explosive work. Strength still climbs via TM increases — it just stops trying to kill you.
2. **Explosive work is distributed, not crammed into one "Oly day."** Snatch, cleans, and plyos each get a fresh home so they stay high-quality and you touch them often (right for "mixed" Oly competency: cleans loaded, snatch as skill).

## The Week

| Day | Explosive opener (fresh, first) | Main (5's PRO) | Pull priority | Core |
|-----|--------------------------------|----------------|---------------|------|
| **Mon** | **Snatch — skill**, light ~15 min | **Squat** | — | Ab-wheel rollout |
| **Tue** | **Upper plyo** (med-ball / plyo pushups) | **Bench** + OHP | **Pullups** | Hanging leg raises |
| **Wed** | — rest from lifting — | — | — | — · **8–12km easy** |
| **Thu** | (opt. light high-pulls) | **Deadlift** | **Pullups** (primary volume) | Suitcase carry |
| **Fri** | **Cleans (loaded) + lower-body jumps** — the athletic centerpiece | light squat (speed) | — | — (cleans cover trunk) |
| **Sat** | **Upper plyo** (slams / med-ball) | **Bench volume** + OHP | **Pullups** | Side plank |
| **Sun** | — rest from lifting — | — | — | — · **10–15km long slow** |

**Every lift day runs in this order:** day-specific warm-up → **explosive opener (while fresh)** → main lift (5's PRO) → pull work + laterals/face pulls → core finisher → run.

**Placement logic:**
- Snatch is light skill, so it sits before a heavy squat without denting it — and primes the CNS.
- **All high-impact leg/pull power is concentrated on Friday** (cleans + jumps), deliberately *away* from heavy squat (Mon) and heavy deadlift (Thu). Saturday buffers Friday's jumps from Sunday's long run.
- Upper plyos live on both press days because they cost your legs nothing.
- Pullups jump from 2→3 days/week — frequency is what drives them up.

**Rules:**
- Heavy leg days (Mon, Thu) get the **shortest, flattest 3km** — first runs to cut if recovery slips.
- All runs easy/conversational Z2. No hard running on lift days.
- Explosive work is **low-volume, max-intent, full-rest — never a grind.** If bar speed or jump height drops, that set was your last.

## Warm-ups (day-specific — non-optional)

General shape every day: **raise temp (2–3 min easy row/bike/jacks) → day-specific mobility → ramp into the first explosive/main movement.** Explosive days need a more thorough ramp than v4's — you're asking joints to move fast under load.

- **Mon (snatch + squat):** temp raise → ankle rocks, deep-squat holds, wrist prep, band pull-aparts, overhead pass-throughs, hip openers → **empty-bar snatch complex** (muscle snatch → snatch balance → overhead squat) building to your skill weight.
- **Tue (upper plyo + bench):** temp raise → band pull-aparts, arm circles, thoracic openers, cat-cow, **1×10–15 pushups (primer)** → a few light med-ball throws to ramp into the plyo.
- **Thu (deadlift):** temp raise → cat-cow, hip hinges, glute bridges, band good-mornings, hamstring floss → progressively heavier ramp sets to first working weight.
- **Fri (cleans + jumps — the most thorough):** temp raise → ankle/hip/thoracic mobility, **front-rack** stretch, wrist prep, glute activation → **jump ramp: pogos → tuck jumps → box jumps** → **empty-bar clean complex** building to working weight.
- **Sat (upper plyo + bench volume):** same shape as Tue.
- **Running:** easy runs need nothing. Long-run days (Wed/Sun): 60 sec leg swings + ankle circles first.

## Explosive Work (the v5 priority)

All low-volume, maximal intent, full recovery between sets. **Quality is the cap — stop while crisp.**

### Snatch — skill (Mon)
- ~15 min. Empty-to-light complex: muscle snatch → snatch balance → hang snatch → full snatch.
- Working feel **~40–50kg** (technically solid), heavier *only* when every rep's positions are crisp.
- **No load target.** This is motor-skill practice, not a lift to chase.

### Cleans — loaded (Fri)
- Hang/power clean, **5–6 × 2–3 reps.**
- Build to **~50–60kg** (~70–85% of a ~70kg best clean). Sample ramp: 40×3 → 45×3 → 50×2 → 55×2 → 60×2.
- **Governed by bar speed, not %.** The moment the bar slows, that's your top set — stop, even if it's 50.

### Lower-body jumps (Fri)
- Box jumps / broad jumps / short bounds, **3–5 × 3–5.** Land soft, full recovery.
- **Depth jumps only when tendons feel fresh** — auto-regulate against the week's running and heavy-leg load. When in doubt, extensive (lower-impact) over intensive.

### Upper-body plyos (Tue + Sat)
- Med-ball chest pass + overhead throw, or clap/plyo pushups, or slams. **3–5 × 3–5**, explosive, full rest.
- Zero leg cost — that's why they live on press days.

## Progression Engine — 5's PRO

Everything is driven by **Training Max (TM) = 90% of estimated 1RM** for each barbell lift. **Anti-imaginary-numbers rules:** TM is derived from a set you *actually completed* (est. 1RM = weight × (1 + reps/30), Epley), and every working weight rounds **down** to a loadable plate. The calculator does all of this.

### Heavy days — Mon squat, Tue bench, Thu deadlift (5's PRO)

Every top set is a clean **5 reps** — no AMRAP.

| Week | Set 1 | Set 2 | Set 3 (×5) |
|------|-------|-------|------------|
| **1** | 65% ×5 | 75% ×5 | **85% ×5** |
| **2** | 70% ×5 | 80% ×5 | **90% ×5** |
| **3** | 75% ×5 | 85% ×5 | **95% ×5** |
| **4 (deload)** | 40% ×5 | 50% ×5 | 60% ×5 |

The 90%-TM buffer means even 95%×5 is only ~85% of true 1RM — hard, honest, not a grind.

### Starting TMs (derived 2026-07-20)

| Lift | Your set | Est. 1RM | **TM** | Note |
|------|----------|----------|--------|------|
| **Squat** | 90×8 (+3 RIR ≈ 90×11) | ~123 | **110** | Slightly bold; 5's PRO buffer absorbs it |
| **Bench** | 80×4 | ~91 | **80** | Honest |
| **Deadlift** | 100×8 (belt) | ~127 | **105** | ⚠️ Deliberately capped low for your back |

**Deadlift caution:** top set never exceeds ~97.5 vs your 100×8. Position and a pain-free back are the goal; load is secondary. Belt on, brace hard, stop the instant the back talks. Only raise this TM after a full symptom-free cycle.

### Volume / secondary work
- **Fri light squat (speed):** ~65% TM (**~70kg**), 5×3 fast — priming reps after cleans+jumps, never a grind.
- **Sat bench volume:** 5×5 @ ~65% TM (**~52.5kg**, your current easy set). Bump to 55 when it flies.
- **OHP (Tue + Sat):** owned-weight, no %. Start **30kg, 5×10.** When all 50 reps move fast and the last set isn't grindy → 32.5kg. That's the whole rule.

### TM progression (the anti-nightmare knob)
- +2.5kg upper / +5kg lower per **cycle** — but given the scaling worry, bump squat/DL **every *other* cycle.** Slower is the point.
- If a top 5 turns into a grind two cycles running, **reset that lift's TM to 85%** of current and re-climb.

## Pullups — promoted to priority (Tue / Thu / Sat)

The frequency jump (2→3 days) is the whole point — pullups respond to frequency.

- **Weekly total-rep target**, climb ~2–3 reps/week. Start where you are; any set/rep split to hit it. **Thu is your highest-volume day.**
- Antagonist pairing with bench on Tue/Sat means they cost almost nothing.
- Once **3×8 clean bodyweight** is easy → start adding weight and reset the target. Chin-ups count.

## Accessories (kept lean from v4)

Compound-first, **3×8–12** unless noted. After the main lift each day, in order: **pull work → laterals + face pulls → core → run.**

### Rotating compound — pick ONE per day
| Day | Options |
|-----|---------|
| Mon | RDLs, walking lunges |
| Tue | Dips, DB rows |
| Thu | Barbell rows, chins |
| Sat | Dips, rear-delt flies |

*(Fri has no rotating compound — cleans + jumps + light squat are the day.)*

### Every lift day (daily shoulder insurance)
- **Lateral raises** 3×12–15
- **Face pulls** 3×15

## Core (Pallof dropped)

The Olympic lifts already hammer the trunk (bracing, anti-flexion), so total core volume **drops** vs v4 — Friday needs no finisher.

| Day | Movement | Prescription | Function |
|-----|----------|--------------|----------|
| Mon | **Ab-wheel rollout** | 3 sets to controlled failure | Anti-extension |
| Tue | Hanging leg raises | 3×10–15 | Flexion |
| Thu | Suitcase carry | 3×20m/side | Anti-lateral flexion + grip |
| Fri | — (cleans cover it) | — | — |
| Sat | Side plank | 3×30–45s/side | Anti-lateral flexion |

- Ab-wheel gets the marquee Monday slot (your favorite).
- **Deload week:** 1 set each, or skip.

## Running (unchanged from v4 — it works)

All easy Z2. No hard running on lift days.

| Day | Distance |
|-----|----------|
| Mon / Thu | 3km (heavy legs — flat, truly easy) |
| Tue / Fri / Sat | 4–5km |
| Wed | 8–12km easy |
| Sun | 10–15km long slow |

- Weekly total ≈ **30–45km**. Build Sunday's long run by ~1km/week, cap, then hold.
- **Deload week: cut all runs ~40%** so the whole system recovers together.

---

## Recovery Notes

- No fully-rest day — sustainable **only** if easy runs stay genuinely easy and explosive work stays low-volume.
- The 5/3/1 deload (every 4th week) + the run cutback keeps this from grinding you down. Don't skip it.
- **First knobs to pull if fatigue accumulates**, in order: (1) Fri jump volume, (2) Mon/Thu run distance, (3) volume-day accessory sets, (4) an extra deload.
- The back is the canary: if deadlift or cleans provoke it, cap load and drop TM before pushing on.
